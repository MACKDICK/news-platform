# Decisiones Tecnológicas

## Backend

### Laravel

Seleccionado por:

* Ecosistema maduro
* Soporte para API REST
* Middleware
* Testing integrado

## Frontend

### React

Seleccionado por:

* Componentización
* Comunidad amplia
* Integración sencilla con APIs

## Mobile

### React Native

Seleccionado por:

* Reutilización de conocimientos React
* Desarrollo Android rápido

## Base de Datos

### MySQL

Seleccionado por:

* Motor relacional estable
* Compatibilidad con Laravel

## Autenticación

### JWT

Seleccionado por:

* Arquitectura Stateless
* Escalabilidad
* Compatibilidad con aplicaciones móviles

## Contenedores

### Docker

Seleccionado para garantizar portabilidad y consistencia entre entornos.
