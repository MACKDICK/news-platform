# Visión del Producto

## Nombre del Proyecto

News Platform

## Objetivo

Desarrollar una plataforma digital de noticias que permita a los usuarios acceder a información actualizada mediante una experiencia moderna, intuitiva y segura, disponible desde aplicación web y aplicación móvil Android.

## Problema a Resolver

Los usuarios necesitan acceder rápidamente a noticias organizadas por categorías, con una navegación sencilla y segura mediante autenticación.

## Solución

La solución consistirá en una API REST desarrollada en Laravel, protegida mediante JWT, consumida por una aplicación web desarrollada en React y una aplicación móvil Android desarrollada con React Native.

## Beneficios

* Acceso seguro mediante JWT.
* Inicio de sesión tradicional y social.
* Consulta de noticias y categorías.
* Noticias relacionadas.
* Arquitectura desacoplada.
* Escalabilidad.
* Compatibilidad web y móvil.
